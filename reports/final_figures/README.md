# Final ERP Figures

This folder contains final, report-ready ERP figures generated by:

```bash
python scripts/03e_make_final_erp_figures.py --config configs/config.yaml
```

## Figures

1. `figure_01_face_nonface_difference_waves`
   - 2x2 panel of Face - Non-face difference waves.
   - Key ROIs: scalp late ROI, scalp face ROI, strict-ear, right-ear.

2. `figure_02_subject_level_w4_boxplot`
   - Subject-level W4 distributions for key ROIs.

3. `figure_03_w4_w5_summary`
   - W4 vs W5 descriptive summary.

4. `figure_04_w4_w5_paired_subjects`
   - Paired subject-level W4-to-W5 changes.

## Tables

- `final_figure_source_values.csv`
  - Source values used in the difference-wave figure.

- `final_w4_w5_paired_stats.csv`
  - Paired subject-level W4-W5 statistics.

## Scientific framing

The main result should be framed as a Face - Non-face stimulus-locked ERP effect,
not as participant-level emotion recognition.
